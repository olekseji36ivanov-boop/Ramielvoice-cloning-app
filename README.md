# Ramiel Voice Cloning Application

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/olekseji36ivanov-boop/Ramielvoice-cloning-app.git
   cd Ramielvoice-cloning-app
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Download the required models:
   - Follow the instructions in the [models folder](models/README.md).

## Usage Guide
1. Start the application:
   ```bash
   python app.py
   ```
2. Use the web interface at `http://localhost:5000` to convert text to speech.
